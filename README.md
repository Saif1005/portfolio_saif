## Déploiement sur GitHub Pages

### 1. Créer le dépôt GitHub

```bash
cd portfolio_saif
git init
git add .
git commit -m "Portfolio professionnel — Saif Akkari"
git branch -M main
git remote add origin https://github.com/Saif1005/portfolio_saif.git
git push -u origin main
```

### 2. Activer GitHub Pages

1. Allez sur **Settings** → **Pages** dans votre dépôt GitHub
2. Sous **Source**, sélectionnez **Deploy from a branch**
3. Choisissez la branche **main** et le dossier **/ (root)**
4. Cliquez **Save**

Votre site sera accessible à : `https://saif1005.github.io/portfolio_saif/`

### 3. Domaine personnalisé (optionnel)

Ajoutez un fichier `CNAME` à la racine avec votre domaine, puis configurez les DNS chez votre registrar.
